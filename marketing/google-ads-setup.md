## 7. Conversion tracking (free — no GTM)

**Option A — Simple (recommended to start):**

1. Google Ads → Goals → Conversions → New conversion action
2. Category: **Submit lead form** or **Book appointment**
3. Source: **Website**
4. Enter URL: `cal.com/iliana-valiullina`
5. Match: **URL contains** `cal.com/iliana-valiullina`
6. Count: One
7. Install the Google tag if prompted (one snippet — or use Google tag only on /lessons/ page)

**Option B — Even simpler for week 1:**

Track **clicks to Cal.com** from the landing page using a conversion action on outbound link clicks (Google Ads can set up "click" conversion on thank-you page on Cal.com after booking if Cal.com provides confirmation URL).

**Also free:** Google Ads dashboard shows clicks, CTR, cost — even without conversion tracking.
